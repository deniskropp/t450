# Contributing to t450

This is a personal experimental workspace for meta-infrastructure, KickLang, and agent experiments.

## How to Contribute (Internal / Personal)

- All changes should be made via feature branches.
- Use clear, TAS-aligned commit messages when possible.
- New Agent Skills must pass relevant validation (Google AI Edge Gallery, MCP, etc. as applicable).
- Updates to specs should maintain both human-readable and `.kl.md` editions when present.

## Questions

Open an issue or discuss in the relevant KickLang / MetaForge / Aetheria sessions.
